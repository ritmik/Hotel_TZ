Тестовое задание [Android]

Необходимо написать приложение, которое загружает с сервера JSON-документ с информацией о нескольких отелях (API 1.1), парсит его и выводит на экран данные этих отелей. Формат отображения этих данных остается на усмотрение разработчика. Пока идет загрузка данных, пользователь должен об этом знать, соответственно, необходимо отображать progress indicator.

Должна присутствовать возможность отсортировать отели по одному из двух параметров: по расстоянию отеля от центра города или по количеству свободных номеров (данные есть в JSON-документе). Контрол для выбора типа сортировки остаётся на усмотрение разработчика. Первичная сортировка должна совпадать с выдачей с сервера.

Также необходимо предусмотреть возможность просмотра подробной информации, включая изображение (API 1.3), об отеле на отдельном экране. Данные конкретного отеля можно получить по URL, составленному следующим образом (API 1.2, ID каждого отеля указан в исходном JSON-документе). Формат отображения данных об отеле остаётся на усмотрение разработчика.

Приложение должно поддерживать систему Android 5.0 и выше. Ориентации — портретная и ландшафтная. Важно: обязательно соблюдение гайдлайнов и использование стандартных контролов (экшнбар, спиннеры, кнопки). Язык — Kotlin.
